# Static HTML/CSS Copy

This is a portable HTML/CSS copy of the confirmed Ordo Studio UXI. Use it as
the static snapshot reference for user-friendly Ordo surfaces.

- `index.html` contains the static structure.
- `styles.css` contains the visual language, layout, cards, sidebar, top signals, and composer styling.

The full interactive React source remains one folder up in `src/`.

When building app or platform UXI, match this snapshot's operator-first
principles: visible controls, unified tabs, readable status, surfaced logs, no
overlapping scroll panes, and no bland default coder UI.
