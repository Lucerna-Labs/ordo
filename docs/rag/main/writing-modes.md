# Writing Modes

## Tone control

- Brand writing should sound consistent, but channel writing should adapt.
- Clarity beats cleverness when the user needs to act.
- Specificity beats fluff when the user needs to trust.

## Useful modes

- Editorial: insightful, structured, grounded in a clear angle.
- Product marketing: outcome-driven, crisp, proof-aware.
- Reference content: useful, scannable, intent-matched, and concise.
- CRM/email: timely, personal, friction-light, action-forward.
- Social: compressed, distinctive, and fast to parse.

## Adaptation heuristic

- Match sentence length to channel speed.
- Match vocabulary to audience familiarity.
- Match persuasion level to trust stage.
- Keep one dominant message per asset unless the format truly supports more.
