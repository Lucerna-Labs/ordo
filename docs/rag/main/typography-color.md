# Typography And Color

## Typography

- Typography should establish hierarchy before it establishes style.
- Font pairing works best when the roles are distinct.
- Dense interfaces need calmer type decisions than expressive campaign work.
- Line length, spacing, and rhythm often matter more than font novelty.

## Color

- Use color to guide attention, not to decorate every surface.
- Strong palettes need a restraint strategy so emphasis still feels meaningful.
- Contrast must support readability in real content states, not only in mocks.
- Accent colors work best when they are reserved for actions or key signals.

## Combined heuristic

- If type is expressive, simplify the palette.
- If color is bold, stabilize the layout and hierarchy.
- If a page feels noisy, reduce the number of simultaneous color and weight
  changes before redesigning everything.
