# Design Advanced

## System-level thinking

- Good design systems balance consistency with deliberate moments of surprise.
- Reusable patterns should preserve intent, not flatten every page into the same
  shape.
- Motion, pacing, and reveal order matter as much as static layout.

## Advanced heuristics

- Build tension with contrast in density, scale, or rhythm.
- Use asymmetry when it creates direction, not when it creates confusion.
- Let supporting elements create atmosphere while the content stays legible.
- Brand expression should show up in structure, not only in surface styling.

## Review checklist

- Does the layout still work when content expands?
- Does the visual language still feel intentional on small screens?
- Does the page have a clear emotional tone beyond "clean"?
- Is the distinctive element helping the message or distracting from it?
