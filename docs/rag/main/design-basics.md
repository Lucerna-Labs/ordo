# Design Basics

## Core principles

- Start with hierarchy before decoration.
- Make one thing obviously primary.
- Reduce competing focal points.
- Use repetition for consistency and contrast for emphasis.
- Leave enough empty space for the important element to breathe.

## Composition basics

- Group related elements tightly and separate unrelated groups clearly.
- Align elements to a visible structure instead of letting them float.
- Use scale changes to show importance instead of relying only on color.
- Keep scanning paths obvious on both desktop and mobile.

## Practical heuristic

- If a layout feels messy, remove one idea before adding another.
- If everything feels equally loud, strengthen the headline and soften the rest.
- If a design looks generic, sharpen the point of view in typography, shape, or
  framing rather than piling on more components.
