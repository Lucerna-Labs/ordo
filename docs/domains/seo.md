# SEO Domain

The `seo.*` domain owns search-oriented packaging, metadata, and readiness
checks.

## Scope

- SEO title and description packaging
- slug guidance
- search intent framing
- internal-link notes
- metadata audits
- launch-readiness checks for search surfaces

## Future capabilities

- `seo.package_metadata`
  - prepare titles, descriptions, slugs, and metadata notes
- `seo.audit_readiness`
  - validate whether a content item satisfies the required SEO checklist
- `seo.generate_internal_links`
  - propose related-link opportunities and anchor guidance
- `seo.extract_search_intent`
  - summarize the search angle a piece is trying to satisfy

## Inputs this domain cares about

- target keyword or topic
- search intent
- page purpose
- SERP positioning notes
- metadata constraints
- related content inventory

## Outputs this domain should produce

- metadata packages
- SEO checklists
- readiness findings
- internal-link guidance
- search-intent summaries

## Boundaries

- creative brief and asset work belong to `creative.*`
- approvals and routing belong to `workflow.*`
- field mapping and publish payloads belong to `cms.*`
